C Programming Tutorials

Introduction to C Programming

C is a powerful and widely used programming language known for its efficiency, flexibility, and portability. Developed in the early 1970s by Dennis Ritchie at Bell Labs, C has had a profound impact on the field of computer programming and has influenced many modern programming languages.

What is C?

C is a general-purpose, procedural programming language that provides low-level access to system memory and hardware. It was designed to be compiled into efficient machine code, making it suitable for developing system software, such as operating systems, device drivers, and embedded systems.

Compilers for C

To write and execute C programs, you need a C compiler, which translates the human-readable C code into machine-readable instructions that can be executed by a computer. There are several popular C compilers available, including:

  • MinGW (Recommended)
  • gcc: The GNU Compiler Collection (GCC) is a widely used open-source compiler suite that supports various programming languages, including C.
  • clang: Clang is another open-source compiler suite developed by LLVM project, known for its fast compilation and modern language features.
  • Microsoft Visual C++: Visual C++ is a compiler provided by Microsoft for developing applications on Windows platforms.

History of C

C was developed by Dennis Ritchie at Bell Labs in the early 1970s as an evolution of the B programming language. It was initially used to implement the UNIX operating system, which was written in C, making C one of the first programming languages used for system programming. C gained popularity due to its simplicity, efficiency, and ability to access hardware directly, leading to its widespread adoption in the computing industry.

Basic Structure and Syntax of C

In C programming, a program consists of one or more functions, where each function contains a set of statements that perform specific tasks. Let's explore the basic structure and syntax of C:

1. Structure of a C Program

A typical C program consists of:

  • Preprocessor Directives: These are instructions to the compiler that begin with a '#' symbol. They are used to include header files, define macros, etc.
  • Function Definitions: C programs contain one or more functions. The 'main()' function is the entry point of a C program.
  • Statements: Each function contains a set of statements that specify the actions to be performed.

2. Syntax of a C Program

The syntax of a C program includes:

  • Variables: Variables are used to store data. They must be declared before they are used and follow certain naming conventions.
  • Constants: Constants are fixed values that do not change during program execution.
  • Comments: Comments are used to document code and improve readability. They can be single-line or multi-line comments.
  • Statements: Statements are instructions that perform specific actions. They end with a semicolon (;).
  • Functions: Functions are blocks of code that perform a specific task. They have a name, parameters (if any), and a return type.
  • Control Structures: Control structures such as if-else, switch-case, loops (while, do-while, for) are used to control the flow of execution in a program.

Here's a simple example of a C program:

#include

int main() {
 // Print "Hello, World!" to the console
 printf("Hello, World!");
 return 0;
}

Comments and Variables in C

Comments and variables are essential components of any programming language, including C. Let's explore their usage:

1. Comments

Comments are used to add explanatory notes within the source code. They are ignored by the compiler and serve to improve code readability and maintainability. C supports two types of comments:

  • Single-line comments: Denoted by two forward slashes (//). They comment out a single line of code.
  • Multi-line comments: Enclosed between /* and */, allowing comments to span multiple lines.

2. Variables

Variables are named storage locations used to store data during program execution. In C, variables must be declared before they can be used. Each variable has a data type that determines the type of data it can store.

Common data types in C include:

  • int: Integer data type for storing whole numbers.
  • float: Floating-point data type for storing decimal numbers.
  • char: Character data type for storing single characters.
  • double: Double precision floating-point data type.

Variables are declared using the syntax: data_type variable_name;

Example:

Here's an example demonstrating the usage of comments and variables in C:

#include

int main() {
 // Declare variables
 int num1 = 10; // Integer variable
 float num2 = 3.14; // Float variable
 char ch = 'A'; // Character variable

 // Print variable values
 printf("num1: %d\n", num1);
 printf("num2: %f\n", num2);
 printf("ch: %c\n", ch);

 return 0;
}

Data Types in C

Data types specify the type of data that variables can store. In C, data types can be classified into several categories, including:

1. Basic Data Types

Basic data types represent fundamental values and are directly supported by the C language. Common basic data types include:

  • int: Integer data type for storing whole numbers.
  • float: Floating-point data type for storing decimal numbers.
  • char: Character data type for storing single characters.
  • double: Double precision floating-point data type.

2. Derived Data Types

Derived data types are derived from basic data types and are used to represent complex data structures. Common derived data types include:

  • Array: A collection of elements of the same data type stored in contiguous memory locations.
  • Pointer: A variable that stores the memory address of another variable.
  • Structure: A user-defined data type that groups together variables of different data types under a single name.
  • Union: Similar to a structure, but all members share the same memory location.

3. Enumeration Data Type

Enumeration data type allows you to define a set of named integer constants. Each constant is assigned a unique value starting from 0 by default.

4. Void Data Type

The void data type represents the absence of type. It is used to indicate that a function does not return any value or to declare generic pointers.

Example:

Here's an example demonstrating the usage of various data types in C:

#include

int main() {
 // Declare variables of different data types
 int num = 10;
 float pi = 3.14;
 char ch = 'A';
 double salary = 50000.50;

 // Print variable values
 printf("Integer: %d\n", num);
 printf("Float: %f\n", pi);
 printf("Character: %c\n", ch);
 printf("Double: %lf\n", salary);

 return 0;
}

Constants in C

In C programming, constants are fixed values that do not change during program execution. They can be classified into two main categories:

1. Numeric Constants

Numeric constants are fixed numeric values that are directly specified in the source code. They can be of different types:

  • Integer Constants: Whole numbers without any decimal point, such as 10, -15, 0.
  • Floating-point Constants: Numbers with a decimal point or exponential notation, such as 3.14, -0.5, 1.2e3.
  • Character Constants: Single characters enclosed in single quotes, such as 'A', 'b', '5'.
  • String Constants: Sequences of characters enclosed in double quotes, such as "Hello", "C programming".

2. Symbolic Constants

Symbolic constants are identifiers that represent fixed values throughout the program. They are declared using the #define preprocessor directive or the const keyword.

Using #define:

#define PI 3.14

Using const keyword:

const int MAX_SIZE = 100;

Example:

Here's an example demonstrating the usage of numeric and symbolic constants in C:

#include

#define PI 3.14159
const int MAX_VALUE = 100;

int main() {
 printf("Value of PI: %f\n", PI);
 printf("Maximum value: %d\n", MAX_VALUE);

 return 0;
}

Operators in C

Operators are symbols used to perform operations on variables and values in a C program. Let's explore the different types of operators:

1. Arithmetic Operators

Arithmetic operators are used to perform mathematical operations on numeric operands.

  • + (Addition): Adds two operands.
  • - (Subtraction): Subtracts the second operand from the first.
  • * (Multiplication): Multiplies two operands.
  • / (Division): Divides the first operand by the second.
  • % (Modulus): Returns the remainder of the division of the first operand by the second.

Example:

int a = 10;
int b = 5;
int sum = a + b; // sum = 15
int difference = a - b; // difference = 5
int product = a * b; // product = 50
int quotient = a / b; // quotient = 2
int remainder = a % b; // remainder = 0
printf("Sum: %d\n", sum);
printf("Difference: %d\n", difference);
printf("Product: %d\n", product);
printf("Quotient: %d\n", quotient);
printf("Remainder: %d\n", remainder);
 Sum: 15
Difference: 5
Product: 50
Quotient: 2
Remainder: 0

2. Relational Operators

Relational operators are used to compare the values of operands.

  • == (Equal to): Returns true if the operands are equal.
  • != (Not equal to): Returns true if the operands are not equal.
  • < (Less than): Returns true if the first operand is less than the second.
  • > (Greater than): Returns true if the first operand is greater than the second.
  • <= (Less than or equal to): Returns true if the first operand is less than or equal to the second.
  • >= (Greater than or equal to): Returns true if the first operand is greater than or equal to the second.

Example:

int x = 5;
int y = 10;
if (x == y) {
 printf("x is equal to y");
} else {
 printf("x is not equal to y");
} x is not equal to y

3. Logical Operators

Logical operators are used to perform logical operations on boolean operands.

  • && (Logical AND): Returns true if both operands are true.
  • || (Logical OR): Returns true if at least one operand is true.
  • ! (Logical NOT): Returns true if the operand is false (negation).

Example:

int a = 5;
int b = 10;
if (a > 0 && b > 0) {
 printf("Both a and b are positive");
}
if (a > 0 || b > 0) {
 printf("At least one of a or b is positive");
}
if (!(a > 0)) {
 printf("a is not positive");
} Both a and b are positive
At least one of a or b is positive

4. Assignment Operators

Assignment operators are used to assign values to variables.

  • = (Simple assignment): Assigns the value of the right operand to the left operand.
  • += (Add and assign): Adds the value of the right operand to the left operand and assigns the result to the left operand.
  • -= (Subtract and assign): Subtracts the value of the right operand from the left operand and assigns the result to the left operand.
  • *= (Multiply and assign): Multiplies the value of the right operand by the left operand and assigns the result to the left operand.
  • /= (Divide and assign): Divides the value of the left operand by the right operand and assigns the result to the left operand.
  • %= (Modulus and assign): Computes the modulus of the left operand with the right operand and assigns the result to the left operand.

Example:

int x = 10;
x += 5; // Equivalent to x = x + 5
printf("Value of x after addition: %d\n", x);
x *= 2; // Equivalent to x = x * 2
printf("Value of x after multiplication: %d\n", x); Value of x after addition: 15
Value of x after multiplication: 30

5. Bitwise Operators

Bitwise operators are used to perform bitwise operations on integer operands.

  • & (Bitwise AND): Performs a bitwise AND operation between the bits of the operands.
  • | (Bitwise OR): Performs a bitwise OR operation between the bits of the operands.
  • ^ (Bitwise XOR): Performs a bitwise XOR (exclusive OR) operation between the bits of the operands.
  • ~ (Bitwise NOT): Performs a bitwise NOT (complement) operation on the operand.
  • << (Left shift): Shifts the bits of the left operand to the left by the number of positions specified by the right operand.
  • >> (Right shift): Shifts the bits of the left operand to the right by the number of positions specified by the right operand.

Example:

int a = 5; // Binary representation: 0101
int b = 3; // Binary representation: 0011
int bitwise_and = a & b; // Result: 0001 (1 in decimal)
int bitwise_or = a | b; // Result: 0111 (7 in decimal)
int bitwise_xor = a ^ b; // Result: 0110 (6 in decimal)
int bitwise_not_a = ~a; // Result: 11111010 (-6 in decimal)
int left_shift = a << 1; // Result: 1010 (10 in decimal)
int right_shift = a >> 1; // Result: 0010 (2 in decimal) Bitwise AND: 1
Bitwise OR: 7
Bitwise XOR: 6
Bitwise NOT of a: -6
Left shift: 10
Right shift: 2

Format Specifiers in C

Format specifiers are used in C programming to specify the data type of variables that are being printed or scanned. They are placeholders that tell the printf and scanf functions how to interpret the corresponding arguments.

1. Integer Format Specifiers

For printing and scanning integer values:

  • %d: Signed integer (decimal).
  • %u: Unsigned integer (decimal).
  • %x: Unsigned integer (hexadecimal, lowercase).
  • %X: Unsigned integer (hexadecimal, uppercase).
  • %o: Unsigned integer (octal).

Example:

int num = 42;
printf("Decimal: %d\n", num);
printf("Hexadecimal (lowercase): %x\n", num);
printf("Octal: %o\n", num); Decimal: 42
Hexadecimal (lowercase): 2a
Octal: 52

2. Floating-Point Format Specifiers

For printing and scanning floating-point values:

  • %f: Floating-point number (decimal notation).
  • %e: Floating-point number (scientific notation, lowercase).
  • %E: Floating-point number (scientific notation, uppercase).
  • %g: Floating-point number (compact decimal or scientific notation, lowercase).
  • %G: Floating-point number (compact decimal or scientific notation, uppercase).

Example:

float f = 3.14;
printf("Floating-point: %f\n", f);
printf("Scientific notation (lowercase): %e\n", f);
printf("Scientific notation (uppercase): %E\n", f); Floating-point: 3.140000
Scientific notation (lowercase): 3.140000e+00
Scientific notation (uppercase): 3.140000E+00

3. Character Format Specifiers

For printing and scanning character values:

  • %c: Single character.
  • %s: String of characters (null-terminated).

Example:

char ch = 'A';
printf("Character: %c\n", ch);
char str[] = "Hello";
printf("String: %s\n", str); Character: A
String: Hello

4. Pointer Format Specifier

For printing and scanning memory addresses:

  • %p: Memory address (hexadecimal).

Example:

int var = 10;
printf("Address of var: %p\n", &var); Address of var: 0x7ffee9b3a9ec

User Input and Output in C

In C programming, user input and output are handled using the standard input (stdin) and standard output (stdout) streams. The scanf function is used for reading input from the user, while the printf function is used for displaying output to the user.

1. Reading Input from the User

The scanf function is used to read input from the user. It takes format specifiers as arguments to determine the data type of the input being read.

Example:

int num;
printf("Enter an integer: ");
scanf("%d", &num); // Read an integer from the user
printf("You entered: %d\n", num);

Output:

Enter an integer: (user input)
You entered: (value entered by the user)

2. Displaying Output to the User

The printf function is used to display output to the user. It takes format specifiers and corresponding values as arguments to format and print the output.

Example:

int a = 10;
float b = 3.14;
printf("The value of a is %d and the value of b is %f\n", a, b);

Output:

The value of a is 10 and the value of b is 3.140000

Conditional Statements in C

1. If-Else Statement

The if-else statement is used to execute a block of code based on the evaluation of a condition. If the condition evaluates to true, the code inside the if block is executed; otherwise, the code inside the else block (if present) is executed.

Example:

int num = 10;
if (num > 0) {
 printf("Positive number");
} else {
 printf("Negative number or zero");
}

Output:

Positive number

2. Switch-Case Statement

The switch-case statement is used to execute one of several blocks of code based on the value of a variable or an expression. It provides an alternative to nested if-else statements.

Example:

int choice;
printf("Enter a choice (1, 2, or 3): ");
scanf("%d", &choice);
switch (choice) {
 case 1:
  printf("You chose option 1");
  break;
 case 2:
  printf("You chose option 2");
  break;
 case 3:
  printf("You chose option 3");
  break;
 default:
  printf("Invalid choice");
}

Output:

Enter a choice (1, 2, or 3): (user input)
(corresponding message based on user input)

Iteration Statements in C

Loops

Loops are used in programming to execute a block of code repeatedly as long as a specified condition is true. In C, there are different types of loops available:

1. While Loop

The while loop is used to repeatedly execute a block of code as long as the specified condition evaluates to true.

Example:

int i = 1;
while (i <= 5) {
 printf("%d ", i);
 i++;
}

Output:

1 2 3 4 5

2. Do-While Loop

The do-while loop is similar to the while loop, but the condition is evaluated after the block of code is executed, so the block of code is always executed at least once.

Example:

int i = 1;
do {
 printf("%d ", i);
 i++;
} while (i <= 5);

Output:

1 2 3 4 5

3. For Loop

The for loop is used to repeatedly execute a block of code for a specified number of times.

Example:

for (int i = 1; i <= 5; i++) {
 printf("%d ", i);
}

Output:

1 2 3 4 5

4. Nested Loops

Nested loops are loops within loops. They can be used to implement complex logic such as iterating over a 2D array.

Example:

for (int i = 1; i <= 3; i++) {
 for (int j = 1; j <= 3; j++) {
  printf("%d ", j);
 }
 printf("\n");
}

Output:

1 2 3
1 2 3
1 2 3

Break Statement

The break statement is used to exit the loop prematurely, regardless of whether the loop condition is still true.

Example:

for (int i = 1; i <= 5; i++) {
 if (i == 3) {
  break;
 }
 printf("%d ", i);
}

Output:

1 2

Continue Statement

The continue statement is used to skip the current iteration of the loop and continue with the next iteration.

Example:

for (int i = 1; i <= 5; i++) {
 if (i == 3) {
  continue;
 }
 printf("%d ", i);
}

Output:

1 2 4 5

Arrays in C

An array is a collection of elements of the same data type stored in contiguous memory locations. It allows you to store multiple values of the same type under a single name.

1. Array Basics

To declare an array in C, you specify the type of elements it will hold and the number of elements enclosed in square brackets []:

int numbers[5]; // Declares an integer array with 5 elements
float prices[10]; // Declares a float array with 10 elements

You can initialize an array during declaration:

int numbers[5] = {1, 2, 3, 4, 5}; // Initializes an array with specified values
char vowels[] = {'a', 'e', 'i', 'o', 'u'}; // Initializes an array without specifying the size

2. Accessing Array Elements

Array elements are accessed using an index. The index starts from 0 for the first element and increments by 1 for each subsequent element:

int numbers[5] = {1, 2, 3, 4, 5};
printf("First element: %d\n", numbers[0]); // Accesses the first element
printf("Third element: %d\n", numbers[2]); // Accesses the third element

Output:

First element: 1
Third element: 3

3. Array Operations

Arrays support various operations such as traversal, insertion, deletion, and searching.

  • Traversal: Iterating through each element of the array.
  • Insertion: Adding an element at a specific position or at the end of the array.
  • Deletion: Removing an element from a specific position or based on its value.
  • Searching: Finding the index of a specific element in the array.

Strings in C

In C, a string is an array of characters terminated by a null character ('\0'). Strings are used to represent sequences of characters, such as words or sentences.

1. String Basics

To declare and initialize a string in C, you can use a character array:

char greeting[6] = {'H', 'e', 'l', 'l', 'o', '\0'}; // Declares and initializes a string
char name[] = "John"; // Declares and initializes a string without specifying the size

Strings can also be initialized using string literals enclosed in double quotes ("").

2. String Functions

C provides several built-in functions for string manipulation:

  • strlen(str): Returns the length of the string.
  • strcpy(dest, src): Copies the contents of one string to another.
  • strcat(dest, src): Concatenates two strings.
  • strcmp(str1, str2): Compares two strings.
  • strchr(str, ch): Returns a pointer to the first occurrence of a character in a string.
  • strstr(str1, str2): Returns a pointer to the first occurrence of str2 in str1.

Pointers in C

A pointer in C is a variable that stores the memory address of another variable. Pointers are used to access and manipulate memory directly, allowing for efficient memory management and advanced data structures.

1. Pointer Basics

To declare a pointer in C, you use the asterisk (*) symbol followed by the variable name:

int *ptr; // Declares a pointer to an integer
float *fptr; // Declares a pointer to a float

To access the value stored at a memory address pointed to by a pointer, you use the dereference operator (*):

int x = 10;
int *ptr = &x; // Pointer ptr points to the address of variable x
printf("Value of x: %d\n", *ptr); // Dereference ptr to get the value of x

2. Pointer Operations

Pointers support various operations such as:

  • Assignment: Assigning the address of a variable to a pointer.
  • Arithmetic: Performing arithmetic operations on pointers.
  • Comparison: Comparing pointers for equality or ordering.

3. Void Pointers

A void pointer (or generic pointer) is a pointer that can point to objects of any data type. It is declared using the void * type:

void *ptr; // Declares a void pointer

Void pointers are commonly used in functions that operate on generic data.

4. Null Pointers

A null pointer is a special pointer that does not point to any memory location. It is represented by the constant NULL or 0:

int *ptr = NULL; // Initializes a pointer to NULL
float *fptr = 0; // Initializes a pointer to NULL using 0

Null pointers are commonly used to indicate that a pointer does not currently point to valid data.

Structures and Unions in C

In C, structures and unions are used to define custom data types that can contain multiple members of different data types. They allow you to group related variables together into a single unit.

1. Structures

A structure is a collection of variables of different data types grouped together under a single name. It is defined using the struct keyword:

struct Person {
 char name[50];
 int age;
 float salary;
};

To access members of a structure, you use the dot (.) operator:

struct Person p;
strcpy(p.name, "John");
p.age = 30;
p.salary = 50000.0;

2. Unions

A union is similar to a structure, but all members share the same memory location. This means that only one member can be active at a time. Unions are defined using the union keyword:

union Data {
 int i;
 float f;
 char c;
};

Accessing members of a union is similar to accessing members of a structure:

union Data d;
d.i = 10;
printf("%d", d.i); // Outputs 10
d.f = 3.14;
printf("%f", d.f); // Outputs 3.14

However, accessing a different member after assigning a value to one member may result in unexpected behavior due to memory sharing.
