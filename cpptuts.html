C++ Programming Tutorials

Introduction to C++

C++ is a powerful and versatile programming language derived from the C programming language. It was developed by Bjarne Stroustrup at Bell Labs in the early 1980s. C++ is widely used in various domains such as system/software development, game development, web development, and scientific computing.

1. History of C++

C++ was created by Bjarne Stroustrup as an extension of the C programming language. Stroustrup began developing C++ in 1979 at Bell Labs while working on his Ph.D. thesis. The first commercial implementation of C++ was released in 1985.

C++ evolved from C with the addition of object-oriented programming features such as classes, inheritance, and polymorphism. Stroustrup's goal was to create a language that combined the efficiency and low-level control of C with the high-level abstractions and flexibility of Simula, a language known for its support for object-oriented programming.

2. Features of C++

C++ introduces several features beyond C, including:

  • Object-Oriented Programming (OOP): C++ supports classes and objects, encapsulation, inheritance, and polymorphism, making it an object-oriented programming language.
  • Template and Generic Programming: C++ supports template functions and classes, enabling generic programming and algorithm development.
  • Standard Template Library (STL): C++ provides a rich collection of classes and functions in the STL for data structures, algorithms, and I/O operations.
  • Exception Handling: C++ supports exception handling mechanisms for robust error handling and recovery.
  • Memory Management: C++ offers manual memory management through features like dynamic memory allocation and deallocation.

3. Hello World Program in C++

Here's a simple "Hello, World!" program in C++:

#include <iostream>
using namespace std;

int main() {
 cout << "Hello, World!" << endl;
 return 0;
}

4. Setting Up a C++ Development Environment

To start programming in C++, you need a text editor or an Integrated Development Environment (IDE) such as Visual Studio, Code::Blocks, or Qt Creator. You also need a C++ compiler like GCC, Clang, or Microsoft Visual C++ Compiler.

5. Compiling and Running C++ Programs

You can compile and run C++ programs using the command line or an IDE. For example, to compile a program named hello.cpp with GCC:

g++ hello.cpp -o hello
./hello

Basic Structure of a C++ Program

A typical C++ program consists of one or more functions. The main function, main(), serves as the entry point of the program. Here's the basic structure of a C++ program:

1. Preprocessor Directives

Preprocessor directives are instructions to the compiler that begin with the # symbol. They are processed before the compilation of the program. Common preprocessor directives include:

  • #include: Includes header files containing declarations of functions and classes.
  • #define: Defines macros for substitution in the code.

2. Namespace Declaration

Namespaces are used to organize code into logical groups and prevent naming conflicts. The using namespace directive is used to declare the namespaces that will be used in the program:

using namespace std;

3. Main Function

The main function, main(), is the starting point of the program execution. It returns an integer value indicating the program's exit status to the operating system:

int main() {
 // Program statements
 return 0; // Indicates successful execution
}

4. Program Statements

The program statements consist of the code that performs the desired functionality of the program. These statements can include variable declarations, function calls, control structures, and other C++ language constructs.

5. Comments

Comments are used to document the code and improve its readability. C++ supports single-line (//) and multi-line (/* */) comments:

// This is a single-line comment
/*
 This is
 a multi-line
 comment
*/

1. Comments

Comments are used to document the code and improve its readability. C++ supports single-line (//) and multi-line (/* */) comments:

2. Variables

A variable is a named storage location in memory used to store data. In C++, variables must be declared before they are used. The basic syntax for variable declaration is:

datatype variableName;
int age; // Declares a variable 'age' of type int
double salary; // Declares a variable 'salary' of type double

3. Variable Initialization

Variables can be initialized at the time of declaration:

int count = 0; // Initializes 'count' to 0
double pi = 3.14159; // Initializes 'pi' to 3.14159

4. Naming Conventions

Variable names in C++ must follow certain rules:

  • Must begin with a letter (uppercase or lowercase) or underscore (_).
  • Subsequent characters can be letters, digits, or underscores.
  • Cannot be a reserved keyword.

It is a good practice to choose meaningful and descriptive variable names to improve code readability.

Data Types and Constants in C++

1. Data Types

C++ provides several built-in data types to represent different types of values. Some common data types include:

  • Integer Types: int, long, short, unsigned int, etc.
  • Floating-Point Types: float, double, long double
  • Character Types: char, wchar_t, char16_t, char32_t
  • Boolean Type: bool

2. Constants

A constant is a value that cannot be modified during program execution. In C++, constants can be defined using the const keyword or using #define preprocessor directive:

const int MAX_SIZE = 100; // Defines a constant MAX_SIZE with value 100
#define PI 3.14159 // Defines a constant PI with value 3.14159

Constants provide meaningful names to values and improve code readability.

Operators in C++

1. Arithmetic Operators

Arithmetic operators are used to perform mathematical operations on operands:

  • + (Addition): Adds two operands.
  • - (Subtraction): Subtracts the second operand from the first.
  • * (Multiplication): Multiplies two operands.
  • / (Division): Divides the first operand by the second.
  • % (Modulus): Returns the remainder of the division.

Example:

int a = 10;
int b = 5;
int sum = a + b; // sum = 15
int difference = a - b; // difference = 5
int product = a * b; // product = 50
int quotient = a / b; // quotient = 2
int remainder = a % b; // remainder = 0

2. Relational Operators

Relational operators are used to compare the values of two operands:

  • == (Equal to): Returns true if operands are equal.
  • != (Not equal to): Returns true if operands are not equal.
  • > (Greater than): Returns true if the left operand is greater than the right.
  • < (Less than): Returns true if the left operand is less than the right.
  • >= (Greater than or equal to): Returns true if the left operand is greater than or equal to the right.
  • <= (Less than or equal to): Returns true if the left operand is less than or equal to the right.

Example:

int x = 10;
int y = 5;
bool result = (x > y); // result = true

3. Logical Operators

Logical operators are used to perform logical operations on boolean operands:

  • && (Logical AND): Returns true if both operands are true.
  • || (Logical OR): Returns true if either operand is true.
  • ! (Logical NOT): Returns true if the operand is false.

4. Bitwise Operators

Bitwise operators are used to perform bitwise operations on integer operands:

  • & (Bitwise AND): Performs a bitwise AND operation.
  • | (Bitwise OR): Performs a bitwise OR operation.
  • ^ (Bitwise XOR): Performs a bitwise XOR (exclusive OR) operation.
  • ~ (Bitwise NOT): Performs a bitwise complement (inverse) operation.
  • << (Left shift): Shifts bits to the left by a specified number of positions.
  • >> (Right shift): Shifts bits to the right by a specified number of positions.

5. Assignment Operators

Assignment operators are used to assign values to variables:

  • = (Simple assignment): Assigns the value of the right operand to the left operand.
  • += (Add and assign): Adds the value of the right operand to the left operand and assigns the result to the left operand.
  • -= (Subtract and assign): Subtracts the value of the right operand from the left operand and assigns the result to the left operand.
  • *= (Multiply and assign): Multiplies the value of the right operand with the left operand and assigns the result to the left operand.
  • /= (Divide and assign): Divides the value of the left operand by the right operand and assigns the result to the left operand.
  • %= (Modulus and assign): Calculates the remainder when the value of the left operand is divided by the right operand and assigns the result to the left operand.

Basic Input and Output (I/O) in C++

1. Output (cout)

The cout object is used to output data to the standard output stream (usually the console). You can use the insertion operator (<<) to display data:

#include <iostream>
using namespace std;

int main() {
 int num = 10;
 cout << "The value of num is: " << num << endl;
 return 0;
}

This will output: The value of num is: 10

2. Input (cin)

The cin object is used to read data from the standard input stream (usually the keyboard). You can use the extraction operator (>>) to get input:

#include <iostream>
using namespace std;

int main() {
 int num;
 cout << "Enter a number: ";
 cin >> num;
 cout << "You entered: " << num << endl;
 return 0;
}

This will prompt the user to enter a number and then display the entered number.

Conditional Statements in C++

1. IF-ELSE Statement

The IF-ELSE statement is used to execute a block of code based on a condition:

int num = 10;
if (num > 0) {
 cout << "Positive number";
} else {
 cout << "Negative number or zero";
}

In this example, if the variable num is greater than 0, it prints "Positive number"; otherwise, it prints "Negative number or zero".

2. SWITCH-CASE Statement

The SWITCH-CASE statement is used to select one of many blocks of code to be executed:

char grade = 'B';
switch (grade) {
 case 'A':
  cout << "Excellent";
  break;
 case 'B':
  cout << "Good";
  break;
 case 'C':
  cout << "Average";
  break;
 default:
  cout << "Fail";
}

In this example, based on the value of the variable grade, it prints the corresponding message (Excellent, Good, Average, or Fail).

Iteration Statements in C++

1. For Loop

The for loop is used to execute a block of code repeatedly for a fixed number of times:

for (int i = 0; i < 5; i++) {
 cout << "Iteration " << i + 1 << endl;
}

This loop will iterate 5 times, printing "Iteration 1", "Iteration 2", ..., "Iteration 5".

2. While Loop

The while loop is used to execute a block of code repeatedly as long as a condition is true:

int i = 0;
while (i < 5) {
 cout << "Iteration " << i + 1 << endl;
 i++;
}

This loop will iterate until the value of i is less than 5, printing "Iteration 1", "Iteration 2", ..., "Iteration 5".

3. Do-While Loop

The do-while loop is used to execute a block of code repeatedly until a condition is false:

int i = 0;
do {
 cout << "Iteration " << i + 1 << endl;
 i++;
} while (i < 5);

This loop will execute the code block at least once and then continue to iterate until the value of i is less than 5, printing "Iteration 1", "Iteration 2", ..., "Iteration 5".

4. Break and Continue Statements

The break statement is used to exit the loop prematurely, while the continue statement is used to skip the current iteration and continue with the next iteration:

for (int i = 0; i < 10; i++) {
 if (i == 5) {
  break; // Exit the loop when i equals 5
 }
 if (i % 2 == 0) {
  continue; // Skip even numbers
 }
 cout << i << endl;
}

This loop will print odd numbers from 0 to 9 and exit when i equals 5.

Arrays in C++

1. Basics of Arrays

An array is a collection of elements of the same data type stored in contiguous memory locations. Each element in the array can be accessed by its index.

int arr[5] = {1, 2, 3, 4, 5}; // Declaring and initializing an array
cout << "First element: " << arr[0] << endl; // Accessing the first element
cout << "Second element: " << arr[1] << endl; // Accessing the second element

This code initializes an integer array arr with 5 elements and accesses the first and second elements using their indices.

2. Array Operations

Common operations performed on arrays include accessing elements, modifying elements, and iterating through the array.

int arr[5]; // Declaring an array
// Initializing array elements
for (int i = 0; i < 5; i++) {
 arr[i] = i + 1;
}
// Printing array elements
cout << "Array elements: " << endl;
for (int i = 0; i < 5; i++) {
 cout << arr[i] << " ";
}

This code initializes an integer array arr with values from 1 to 5 and then prints all the elements of the array.

Structures, Unions, and Enums in C++

1. Structures

A structure is a user-defined data type that allows you to group related variables of different data types together:

struct Point {
 int x;
 int y;
};

This defines a structure named Point with two members: x and y, both of type int.

2. Unions

A union is a special data type that allows storing different data types in the same memory location. Only one member of the union can be accessed at a time:

union Value {
 int intValue;
 float floatValue;
 char charValue;
};

This defines a union named Value that can store an integer, a float, or a character value.

3. Enums

An enumeration (enum) is a user-defined data type used to assign names to integral constants, making the code more readable and maintainable:

enum Weekday {
 Monday,
 Tuesday,
 Wednesday,
 Thursday,
 Friday,
 Saturday,
 Sunday
};

This defines an enum named Weekday with the names of the days of the week as its members.

Functions in C++

1. Parameters and Return Types

A function in C++ can have parameters (inputs) and a return type (output):

int add(int a, int b) {
 return a + b;
}

This defines a function named add that takes two integer parameters a and b and returns their sum.

2. Recursion

Recursion is a technique where a function calls itself to solve smaller instances of the same problem:

int factorial(int n) {
 if (n == 0) {
  return 1;
 } else {
  return n * factorial(n - 1);
 }
}

This defines a recursive function to calculate the factorial of a number n.

3. Function Overloading

Function overloading allows multiple functions with the same name but different parameter lists:

int max(int a, int b) {
 return (a > b) ? a : b;
}

float max(float a, float b) {
 return (a > b) ? a : b;
}

This defines two overloaded functions named max to find the maximum of two integers and two floats, respectively.

Strings in C++

1. Basics of Strings

In C++, a string is a sequence of characters stored in contiguous memory locations. You can use the string class from the standard library to work with strings:

#include <string>
using namespace std;

string str = "Hello, World!"; // Declaring and initializing a string
cout << "String length: " << str.length() << endl; // Getting the length of the string
cout << "First character: " << str[0] << endl; // Accessing the first character

This code initializes a string str with the value "Hello, World!" and demonstrates accessing the length and first character of the string.

2. String Functions

The string class provides various member functions to manipulate strings:

  • append(): Appends a string to the end of another string.
  • substr(): Extracts a substring from a string.
  • find(): Finds the first occurrence of a substring within a string.
  • replace(): Replaces a portion of the string with another string.
  • And many more...
string str1 = "Hello";
string str2 = "World";
str1.append(", ").append(str2);
cout << "Appended string: " << str1 << endl;
cout << "Substring: " << str1.substr(0, 5) << endl;
cout << "Index of 'World': " << str1.find("World") << endl;
str1.replace(6, 5, "Universe");
cout << "Replaced string: " << str1 << endl;

This code demonstrates some of the common string functions provided by the string class.
